# 二、Hexo

# 2.1 出现Spawn Failed报错

首先尝试关机重启

其次删除.deploy_git文件夹

最后实在不行找到.ssh文件，先测试

ssh -T git@github.com

如果出现22端口连接超时，改成443端口即可

Host github.com
  HostName ssh.github.com
  Port 443
  User git
  IdentityFile ~/.ssh/id_rsa
  IdentitiesOnly yes
