Git 基本使用教程

Git 是一个分布式版本控制系统，适用于代码版本管理与团队协作开发。本文将简要介绍 Git 的一些常用操作命令，包括创建仓库、分支管理以及删除文件/分支的方法。

# 一、初始化仓库

git init
  • 在当前目录初始化一个 Git 仓库。
  • 会创建 .git 文件夹作为版本控制目录。

# 二、克隆远程仓库

git clone <仓库地址>
```****

示例：

```bash
git clone https://github.com/username/repo.git

# 三、添加与提交文件

git add <文件或目录>
git commit -m "提交说明"

示例：

git add .
git commit -m "添加初始项目文件"

# 四、查看状态与历史

git status        # 查看当前状态
git log           # 查看提交历史

# 五、分支操作

# 创建新分支

git branch <分支名>

# 切换分支

git switch <分支名>
# 或（旧写法）
git checkout <分支名>

# 创建并切换新分支

git switch -c <分支名>

# 查看所有分支

git branch

# 六、合并分支

git merge <被合并的分支>

注意：需要先切换到目标分支，再合并。

# 七、删除操作

# 删除文件

git rm <文件名>
git commit -m "删除了某个文件"

# 撤销删除（误删恢复）

git checkout -- <文件名>

# 删除本地分支

git branch -d <分支名>
  • -d 是安全删除（合并过的才允许删）
  • -D 是强制删除（无论是否合并）

# 删除远程分支

git push origin --delete <分支名>

# 八、推送与拉取

git push origin <分支名>   # 推送到远程
git pull origin <分支名>   # 拉取最新代码

# 九、查看远程仓库

git remote -v

# 十、总结

Git 是开发者日常工作中不可或缺的工具，掌握它可以大大提升协作效率与版本管理能力。希望本文能帮助你快速上手 Git 的基本操作。

如需进一步学习，可以参考 Pro Git 中文版。
